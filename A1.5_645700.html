EXPLICACION TRES:Aqu se identifican y eliminan los valores atpicos de la variable Faltas mediante el mtodo de Tukey con k=3. Primero se calculan los cuartiles Q1 y Q3 y, a partir de ellos, el rango intercuartlico (IQR). Con este valor se determinan los lmites inferior y superior para definir qu registros se consideran extremos. Despus, se utilizan condiciones lgicas con np.where para identificar a los estudiantes cuyas faltas se encuentran fuera de estos lmites. Estas observaciones se imprimen para analizarlas y, posteriormente, se eliminan del DataFrame. Se considera que las faltas con valores atpicos deben eliminarse porque representan situaciones poco comunes, como estudiantes con un nmero excesivamente alto de inasistencias que no son representativos de la mayora del grupo. Mantener estos valores podra distorsionar el ajuste del modelo de regresin y generar conclusiones sesgadas, por lo que resulta ms adecuado retirarlos del anlisis.

EXPLICACON CUATRO:En este apartado se realiza un anlisis de la correlacin entre variables para identificar problemas de colinealidad, es decir, cuando dos o ms variables estn demasiado relacionadas entre s y pueden aportar informacin redundante al modelo. Primero se calcula la matriz de correlacin absoluta mediante np.corrcoef, transformando todos los valores a flotantes para evitar errores de tipo. Despus se elimina la diagonal de la matriz, ya que representa la correlacin perfecta de cada variable consigo misma. Se busca la correlacin ms alta de la matriz con np.unravel_index, lo que permite identificar qu variables estn ms relacionadas entre s. Adems, se imprime el valor mximo de correlacin y los nombres de las variables implicadas. Para visualizar de manera ms clara la relacin entre todas las variables, se genera un mapa de calor con seaborn.heatmap, lo que facilita observar patrones de alta correlacin. Finalmente, se elimina la variable G2, ya que se detecta que est fuertemente correlacionada con G1 y G3. La decisin de eliminarla responde a la necesidad de evitar problemas de multicolinealidad que afectaran la estabilidad y la interpretacin del modelo de regresin mltiple.
